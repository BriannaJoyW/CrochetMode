Baby Blanket

Materials Needed

  • 800 yards of worsted weight yarn (size 4) *you can choose multiple colors*

  • Size I Hook (5.50mm)

  • Yarn needle

  • Scissors

    • Stitch Tutorials

  • Chain Stitch (CH)
  • Slip Stitch (SS)

  • Single Crochet (SC)

  • Half Double Crochet (HDC)

    • Pattern

  • Beginning: (With starting color) CH 62

  • Row 1: HDC in 3rd CH from hook, HDC in each chain across (60)

  • Row 2: CH 2, HDC in each stitch across (60)

  • Row 3-57: Repeat row 2 (60)

  • Color Changing: You can switch colors at any point you wish in this pattern! For example you could do 5 rows in one color, 8 rows in another color, 12 rows in another, etc! Have fun with it!

  • Border Row 1: SC around the entire blanket, working 3 SC into each corner. SS into the first stitch.

  • Border Row 2: CH 2, HDC around in each stitch, working 3 HDC into each corner. SS into the first stitch.

  • Border Row 3: CH 1, SC around in each stitch, working 3 SC into each corner. SS into the first stitch.

  • Finishing: Cut yarn with a tail, then pull yarn through the stitch that is on your hook. Thred yarn tail into yarn needle and weave in ends.
